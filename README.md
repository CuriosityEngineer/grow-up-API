# Grow Up API
Feather's api using Mongodb for a simple game we build called Grow Up.

  - [Aeshta](https://github.com/CuriosityEngineer)

## Set up
> npm install

### Check if mongodb is running

> ps aux | grep mongo

### If mongodb is not running 
> brew update

> brew install mongodb
